# HANDOFF.md

本文件是 `openclaw_docs` 项目的持续交接记录。
作用是让后续 Agent 不依赖很长的聊天历史，也能快速知道：这个项目是做什么的、已经做到了哪一步、当前正在处理什么。

## 这个文件的功能
- 记录项目目标与边界。
- 记录已经完成的重要工作，避免重复劳动。
- 记录当前仍在推进的事项，方便继续接手。
- 作为多 Agent 协作时的长期项目记忆浓缩层。

## 项目目标
- 将 `https://docs.openclaw.ai/` 的在线文档同步到本仓库。
- 将仓库中的 `.md` 文件作为官网页面的本地镜像维护。
- 保持文档可复查、可修复、可复跑。
